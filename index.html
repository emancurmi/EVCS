Electronic Vehicle Charging Station

Electric Vehicle Charging Station (EVCS) is an application that can be used on desktop, tablets or phones to allow the user to search for a charging station in his location or in an address given by the user.

Quick steps to start

These instructions will get started on how to use this application.

  • Select your Car Brand
  • Select your Model
  • Select your charging method
  • You can enter the Address or Check the box to let the application get your location
  • Click the Blue Button to start search
  • If you ticked the box a message box will ask you for permision to get your current location
  • Click Allow to continue
  • Browse the markers for additional details about the charging spot
